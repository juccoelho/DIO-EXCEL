# DIO-EXCEL
Excel de vendas do Xbox
Este projeto tem como objetivo a criação de um dashboard de vendas no Excel, focado na organização e visualização de dados. A proposta é transformar dados brutos de vendas em informações visuais claras, facilitando a análise do desempenho e apoiando a tomada de decisões.
O repositório contém o arquivo Excel com o dashboard finalizado e esta documentação introdutória para contextualização do desafio.

## Orientações para Entrega
Organizar o repositório do GitHub com:
README.md explicando o projeto, os dados utilizados e instruções para reprodução.
O arquivo Excel (.xlsx) com o dashboard concluído.
Compartilhar o link do repositório GitHub como entrega do desafio.
